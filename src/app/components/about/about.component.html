About

This is a personal project that merges my two favorite things: Coding and Music.

It is a web-based MIDI controller that allows you to play music using your computer's keyboard or a connected MIDI device.

Requirements:
  • Virtual Midi Port (See below)
  • Midi Connected Soundsource (i.e. synth)
  • A virtual Synth Desktop or if Web synth (https://midi.city/)

Resources:

Free/Open-Source Virtual Synths:

  • Surge XT - Open Source Synthesizer
  • LoopMIDI - Virtual MIDI port
  • midi.city - Web-based virtual Synthesizer.
